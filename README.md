# vue-sudoku-solver

Sodoku solver using backtracking search.

### How to use
1. Input the sudoku numbers in the appropriate boxes
2. Click solve

### Install

```bash
git clone https://github.com/EdwardKerckhofZoneDev/vue-sudoku-solver.git
cd vue-sudoku-solver
yarn install
yarn dev
```
